# Queue

* FIFO (First In First Out)
* Used in BFS

```java
import java.util.Queue
import java.util.LinkedList;

// in java, we need to use LinkedList to declare a Queue
Queue<Integer> queue = new LinkedList<>();
Queue<String> queue2 = new LinkedList<>();

// adding items
queue.add(1);
OR
queue.offer(1);
```
