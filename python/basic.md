# Python Basics

## Functions
* Helps to organize the code

How to make a function

Basic Function
```python
  def say_hi():
    print("Hi")
  
  say_hi()
<!--  prints "Hi"  -->
```

Having a parameter
```python
    def say_hi(name):
    print("Hi " + name)
  
  say_hi("James")
<!--  prints "Hi James"  -->
```
