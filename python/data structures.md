# Data Structures

## Tuples
* Type of data structure
* Immutable (Cannot change any of the element once initialized)

How to create a tuple
```python
  coordinates = (4, 5)
  <!--  coordinates[0] = 10 is not allowed  -->
  print(coordinates[0])
  <!--  returns 4  -->
```

List of Tuples
```python
  coord = [(1,2), (3,4), (4,5)]
```

## Lists
* Mutable (can change the elements)

```python
  coordinates = [4, 5]
  coordinates[0] = 10
  print(coordinates[0])
  <!--  returns 10  -->
```
